# Project Discovery Agent

This agent helps formulate and structure new project ideas.

It will:

1. Interview you about your idea
2. Clarify requirements
3. Define the MVP
4. Propose a technical architecture
5. Generate project documentation

Generated output:

/projects/<project-name>/

Files created:

- project_overview.md
- product_requirements.md
- technical_architecture.md
- roadmap.md
- tasks.md
